Data Binding in Angular

1. One Way Data Binding
  • Interpolation (" {{ 'variable' }} ")
    updates the view with components data

    Example:
    {{ " userName : " + userName}}
    isLoggedIn : {{userName ? "Yes Logged In" : "Not Logged In"}}
  • Property Binding (" [property] ")
    binds the value of a component property to an HTML property
    updates the view with component data
    mostly used to set attributes of HTML elements
  • Event Binding ( "(event)" )
    binds an event to a method of the component
    allows component to respond to users action
